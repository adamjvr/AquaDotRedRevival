# AquaDotRed Starter Project

This is the first vertical-slice scaffold for the AquaDot!Red reboot.

Included:
- SwiftUI app shell
- SpriteKit integration
- Procedural maze rendering
- Procedural glowing dots
- Hardcoded prototype level

To use:
1. Open Xcode
2. Create a new iOS/iPadOS SwiftUI app named `AquaDotRed`
3. Replace the generated Swift files with these
4. Run on iPad simulator

Recommended target:
- iPadOS 17+
- "Designed for iPad" enabled for Mac support
